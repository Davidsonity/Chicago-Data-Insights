### Store the datasets in database tables

To analyze the data using SQL, it first needs to be stored in the database.

The following are the steps taken to store the data in the IBM DB2 database
- Open the Db2 console
- Open the LOAD tool
- Select / Drag the .CSV file for the first dataset
- Next create a New Table, and then follow the steps on-screen instructions to load the data. Name the new tables as follows:

1.  **CENSUS_DATA**
2.  **CHICAGO_PUBLIC_SCHOOLS**
3.  **CHICAGO_CRIME_DATA**
